In the competition there were two types of gamemodes, there was manual where you used a controler
to control the robot and there was autonomous mode where you coded the robot to get as many points as possible.
We used Robot C graphical to code the robot, you drag and drop the diffrent commands to code it.
When your done the code you use a cable connected to the robots brain to insert the code into the robot.

We had two blocks of code, one was to knock the extra rings at the end of the arena.
The other block picks up one set of rings of the pole and puts on the opposite pole.
